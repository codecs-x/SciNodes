# `scinodes::i18n`

Tabla de traducciones plana indexada por *key* con
*fallback* en cadena.

## API

```cpp
namespace scinodes::i18n {
    enum class Language { EN, ES };
    bool        loadFromFile(const std::string& path);
    void        setLanguage(Language lang);
    Language    currentLanguage();
    const char* tr(const char* key);
}
```

## Fallback

`tr(key)` devuelve:

1. Texto del idioma activo si la *key* está.
2. Texto en inglés si la *key* falta en el activo.
3. La *key* misma si tampoco hay inglés.

La cadena hace visible inmediatamente qué falta traducir.

## Convención de keys

Jerárquica con puntos:
`menu.file.save`, `workspace.design`, `statusbar.run`,
`port.dc_motor.voltage`, etc.
