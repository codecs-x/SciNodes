# Idiomas (i18n)

El editor incluye una tabla de traducciones que cubre menú
principal, *tabs* de workspaces, *statusbar*, popups y
*port labels* de los nodos.

## Activar español

Variable de entorno al arrancar:

```bash
SCINODES_LANG=es ./SciNodes
```

`SCINODES_LANG` sin set o cualquier otro valor mantiene el
inglés como default.  El cambio en caliente desde un menú
*View → Language* queda como trabajo futuro (el motor ya lo
soporta, solo falta la UI).

## Cobertura

A esta versión, 223 *strings* en `es.json`.  Cuando una
*key* falta en el idioma activo, el editor cae al inglés;
si tampoco está ahí, devuelve la *key* misma — los huecos
son visibles inmediatamente.
